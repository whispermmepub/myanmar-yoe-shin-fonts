GitHub README.md မှာ ဘယ်လိုသုံးရမလဲ

## မြန်မာဖွန့်များ

သင့် GitHub page / blog မှာ မြန်မာဖွန့်တွေ သုံးချင်ရင်:

1. **fonts/** folder ထဲက font file တွေကို copyယူပါ
2. **css/myanmar-fonts.css** ကို link ထည့်ပါ
3. CSS class တွေသုံးပါ (`.font-yoeshin`, `.font-burma026`, etc.)

GitHub Pages မှာ:
- `fonts/` folder ကို repo ထဲထည့်
- CSS file ထဲက url() path ကို ပြင်ပါ
- OR GitHub-flavored CSS ကို inline သုံးပါ

PDF မှာ ဘယ်လိုသုံးရမလဲ (Python fpdf2)

from fpdf import FPDF

pdf = FPDF()
pdf.add_font('YoeShin', '', 'fonts/A10_YoeShin-Regular.ttf')
pdf.add_font('Burma026', '', 'fonts/Burma026-Regular.ttf')
pdf.add_font('PuPu', '', 'fonts/M01_PuPu Bold.ttf')
pdf.add_font('Myittar', '', 'fonts/M03_Myittar-Regular.ttf')
pdf.add_font('MyanmarAyar', '', 'fonts/MyanmarAyarTyepwriter.ttf')

pdf.add_page()
pdf.set_font('YoeShin', '', 14)
pdf.cell(0, 10, 'ယိုးရှင် ဖွန့်ဖြင့် PDF ရေးခြင်း')
pdf.output('output.pdf')

EPUB မှာ ဘယ်လိုသုံးရမလဲ

import ebooklib
from ebooklib import epub

book = epub.EpubBook()
# Font file ကို EPUB ထဲထည့်
with open('fonts/A10_YoeShin-Regular.ttf', 'rb') as f:
    font_data = f.read()

font = epub.EpubItem(
    uid='yoeshin',
    file_name='fonts/A10_YoeShin-Regular.ttf',
    media_type='application/octet-stream',
    content=font_data
)
book.add_item(font)

# CSS ထဲမှာ font-face ထည့်
style = '''
@font-face {
    font-family: 'YoeShin';
    src: url('fonts/A10_YoeShin-Regular.ttf') format('truetype');
}
body { font-family: 'YoeShin', serif; }
'''
css = epub.EpubItem(uid='style', file_name='style/myanmar.css',
                     media_type='text/css', content=style.encode())
